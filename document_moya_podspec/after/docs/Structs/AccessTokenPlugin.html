AccessTokenPlugin

public struct AccessTokenPlugin: PluginType

A plugin for adding bearer-type authorization headers to requests. Example:

Authorization: Bearer <token>

Note

By default, requests to all TargetTypes will receive this header. You can control this behvaior by conforming to AccessTokenAuthorizable.
