ResourceObserver

public protocol ResourceObserver

Something that can observe changes to the state of a Resource. “State” means latestData, latestError, and isLoading.

Any code that wants to display or process a resource’s content should register itself as an observer using Resource.addObserver(...).

  • resourceChanged(_:event:)

    Called when anything happens that might change the value of the reosurce’s latestData, latestError, or isLoading flag. The event explains the reason for the notification.

    Swift

    func resourceChanged(resource: Resource, event: ResourceEvent)
  • resourceRequestProgress(_:progress:) Default implementation

    Receive updates on progress at regular intervals while a request is in progress. Will always receive a call with a value of 1 when the request completes.

    Default Implementation

    Does nothing.
